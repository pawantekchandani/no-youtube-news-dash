### 2. macOS Automation Daemon

#### [NEW] [com.user.newsdash.plist](file:///Users/pawan/Coding%20projects/Pawans%20personal%20news%20app/launchd/com.user.newsdash.plist)
A macOS launchd XML property list registering `newsdash` as a persistent background daemon:
* Automatically loads the service when the user logs in (`RunAtLoad: true`).
* Keeps it running in the background and restarts on crash (`KeepAlive: true`).
* Redirects stdout and stderr to `logs/launchd-stdout.log` and `logs/launchd-stderr.log`.
* Automatically resolves absolute paths to your user directory (`/Users/pawan/...`) and workspace environment variables.
